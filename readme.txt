== Description ==

Honey Code is a Gutenberg Syntax Higlighter Block

= Features =

* Syntax Highlighting in Gutenberg Post/Page Editor.
* Honey Themes that support Fira Code/iScript Font.
* Other great themes for Syntax Highlighting and more coming!

= Current Themes =
* Honey
* Honey Night
* Honey Bright
* Tomorrow
* Tomrrow Night

= Supported Languages =
* Generic
* PHP
* HTML
* SHELL
* SQL
* Python
* Ruby
* CSS
* Java
* JavaScript
* GO
* C#
* C

If you notice any issues with the Syntax Higlighter, please create a ticket on the [github repo](https://github.com/chrisakelley/honey-code/issues).

== Frequently Asked Questions ==

= Does it work with the Classic Editor? =
No, currently there is no support for the classic editor. Syntax Highlighting is only available from the Gutenberg Editor
